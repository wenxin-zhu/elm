### 项目结构

```
+---front end						# 前端项目代码
|   \---elm_demo
|
+---JDBC 							# JDBC项目代码
|   \---elm_admin
|
+---servlet 						# sevlet项目代码
|   +---elm
|   +---elmclient
|   \---node_modules
|
+---README.md 						# 说明文档
|      
\---spring boot 					# spring boot项目代码
	+---dist.zip					# 用于部署的前端打包文件
    +---elmboot						# 前端代码
    +---elmboot-0.0.9-SNAPSHOT.jar	# 用于部署的后端打包文件
    +---elmclient					# 后端代码
    +---history						# 历史版本
    \---node_modules 				# 使用npm运行代码必须的module文件
```
